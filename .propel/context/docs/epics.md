## Epic Description

### EP-TECH: Project Foundation & Infrastructure Setup

**Business Value**: Enables all subsequent development by establishing the project scaffolding, development environment, CI/CD pipeline, and base architecture. Without this foundation, no feature development can proceed.

**Description**: Bootstrap the green-field AI Smart Reminder project with a .NET 8 Web API backend using Clean Architecture (Domain, Application, Infrastructure, Presentation layers), Angular 18+ frontend with modular architecture (Core, Shared, Feature modules), PostgreSQL database provisioning, Docker containerization with Docker Compose orchestration, structured logging via Serilog/Seq, health check endpoints, and RESTful API scaffolding with OpenAPI/Swagger documentation. Establishes coding standards, project structure, and development workflow.

**UI Impact**: Yes

**Screen References**: N/A

**Key Deliverables**:

- .NET 8 Web API project scaffolded with Clean Architecture layer separation
- Angular 18+ project scaffolded with CoreModule, SharedModule, and initial Feature module structure
- PostgreSQL database provisioned with connection configuration
- Docker Compose configuration for local development (API, PostgreSQL, Redis, Seq)
- Serilog structured logging with Seq sink and correlation ID middleware
- ASP.NET Core Health Check endpoints for database, Redis, and external services
- Swagger/OpenAPI documentation generation via Swashbuckle
- Base CI/CD pipeline configuration

**Dependent EPICs**:

- None

### EP-DATA: Core Data & Persistence Layer

**Business Value**: Enables data operations for all feature epics requiring persistence by establishing the entity model, database schema, relationships, and data integrity rules that underpin every feature.

**Description**: Implement the complete Entity Framework Core data layer including all 11 domain entities (User, Task, Category, TaskCategory, Note, Reminder, Notification, AuditLog, RefreshToken, UserProductivityMetrics, AIInteractionLog) with their attribute definitions, relationship mappings, foreign key constraints, cascade rules, global query filters for soft-delete and row-level isolation, UTC timestamp conventions, unique constraints, and EF Core migration infrastructure supporting zero-downtime deployments. Includes seed/mock data for development.

**UI Impact**: No

**Screen References**: N/A

**Key Deliverables**:

- EF Core DbContext with all 11 entity configurations
- Entity type configurations with attribute mappings and data types
- Foreign key relationships with cascade rules (User→Task, Task→Reminder, etc.)
- Many-to-many relationship via TaskCategory join entity
- Global query filters for soft-delete (IsDeleted) and row-level isolation (UserId)
- UTC timestamp storage convention (timestamptz) for all date/time columns
- Unique constraints on User.Email and Category.Name per user
- Initial EF Core migration with version tracking
- Zero-downtime migration support with backward-compatible schema changes
- Seed data for development and testing

**Dependent EPICs**:

- EP-TECH - Foundational - Requires project scaffolding and EF Core setup
