## Brand Guidelines

### Logo

- **Primary**: Indigo wordmark "AI Smart Reminder" with lightning bolt icon
- **Icon only**: Lightning bolt in `color.primary.500` for favicons and mobile nav
- **Minimum size**: 24px icon, 120px wordmark
- **Clear space**: 1x icon height on all sides

### Iconography

- **Style**: Outlined, 1.5px stroke weight
- **Active/Selected**: Filled variant
- **Size**: 16px (inline), 20px (navigation), 24px (primary actions)
- **Color**: Inherits from parent context (text color or semantic color)
- **Source**: Lucide Icons (open-source, consistent with outlined style)

### Illustration Style

- **Type**: Flat, geometric, minimal
- **Color palette**: Primary and secondary tones with neutral accents
- **Usage**: Empty states, onboarding, error pages
- **Characters**: Abstract human figures (inclusive, non-specific)

## Accessibility Requirements

- **WCAG Level**: AA (all screens)
- **Color Contrast**: Minimum 4.5:1 for normal text; 3:1 for large text (18px+ regular or 14px+ bold); 3:1 for UI components
- **Focus States**: 2px outline with `color.primary.500`, 2px offset, minimum 3:1 contrast against adjacent colors
- **Touch Targets**: Minimum 44x44px for all interactive elements on mobile
- **Screen Reader**: Semantic HTML, ARIA labels on icon-only buttons, live regions for dynamic content (toasts, loading states)
- **Motion**: Honor `prefers-reduced-motion: reduce` media query
- **Keyboard**: Full keyboard navigation; Tab order follows reading order; Escape closes modals/drawers; focus trap in modals
