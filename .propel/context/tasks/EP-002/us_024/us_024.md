# User Story - US_024

## Story ID

* ID Format: US_024

## Story Title

* Task Notes CRUD Operations

## Description

* As an **authenticated user**, I want to add, update, and delete text notes on my tasks, so that I can capture additional context and details for each task.

## Acceptance Criteria

* Given an authenticated user has a task, When they add a note with text content, Then the system MUST create the note linked to the task and return 201 Created.
* Given a note exists on a task, When the user updates the note content, Then the system MUST update the content and refresh the UpdatedAt timestamp.
* Given a note exists on a task, When the user deletes the note, Then the system MUST permanently remove the note record.
* Given a task has multiple notes, When the task details are retrieved, Then all associated notes MUST be included in the response, sorted by creation date.

## Edge Cases

* What happens when a user tries to add a note to another user's task? The system MUST return 404 Not Found due to row-level isolation.
* How does the system handle an empty note content? The system MUST return 400 Bad Request with "content is required" validation error.
