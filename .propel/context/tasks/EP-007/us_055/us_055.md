# User Story - US_055

## Story ID

* ID Format: US_055

## Story Title

* Productivity Score Calculation

## Description

* As an **authenticated user**, I want weekly and monthly productivity scores calculated from task completion rate, on-time completion percentage, and reminder response rate, so that I can track my productivity trends over time.

## Acceptance Criteria

* Given a user has task completion history, When the productivity score is calculated, Then it MUST factor in: task completion rate, on-time completion percentage, and reminder response rate.
* Given the score is computed, When stored in UserProductivityMetrics, Then it MUST include PeriodType (Daily/Weekly/Monthly), PeriodStart, PeriodEnd, and ProductivityScore.
* Given a weekly period, When the score is calculated, Then it MUST cover Monday-Sunday of the relevant week.
* Given AI is available, When the score includes AI-weighted factors, Then the AI MUST provide scoring adjustments based on task complexity and behavior patterns (HYBRID: deterministic + AI).

## Edge Cases

* What happens when a user has no completed tasks in a period? The ProductivityScore MUST be 0 with all metric counters at 0.
* How does the system handle partial weeks (e.g., user registered mid-week)? The score MUST be calculated based on available data with a note about the partial period.
