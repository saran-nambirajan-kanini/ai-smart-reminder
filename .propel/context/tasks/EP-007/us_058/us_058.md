# User Story - US_058

## Story ID

* ID Format: US_058

## Story Title

* Angular Analytics Dashboard Module

## Description

* As a **user**, I want an Angular dashboard screen displaying task metrics, completion charts, productivity scores, and AI insights, so that I can monitor my productivity through a visual, data-rich interface.

## Acceptance Criteria

* Given the user navigates to the dashboard (SCR-013), When data is loaded, Then it MUST display key metrics (total, completed, pending, overdue tasks) above the fold without scrolling on desktop (per UXR-104).
* Given metrics are displayed, When charts are rendered, Then task completion rate and missed reminders MUST be shown as time-series charts with period selection (daily/weekly/monthly).
* Given the productivity score is available, When displayed, Then it MUST show the current score with trend indicator (up/down arrow).
* Given AI recommendations are available, When displayed, Then they MUST appear in a dedicated section with accept/dismiss controls.
* Given any dashboard element has loading data, When async fetch is in progress, Then skeleton screens MUST appear within 200ms (per UXR-501).

## Edge Cases

* What happens when the dashboard is viewed on mobile? The layout MUST stack metrics vertically in single-column format with scrollable charts.
* How does the system handle stale data? A "last updated" timestamp MUST be visible indicating when metrics were last refreshed.
