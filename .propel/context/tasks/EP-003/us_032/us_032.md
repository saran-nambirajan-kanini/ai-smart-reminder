# User Story - US_032

## Story ID

* ID Format: US_032

## Story Title

* Upcoming Reminders List Endpoint

## Description

* As an **authenticated user**, I want to view a list of my upcoming reminders sorted by scheduled time, so that I can see what's coming next and plan accordingly.

## Acceptance Criteria

* Given an authenticated user calls the upcoming reminders endpoint, When active reminders exist, Then the system MUST return reminders sorted by NextOccurrence (or ScheduledTime) in ascending order.
* Given reminders are returned, When inspecting each reminder, Then it MUST include: reminder ID, associated task title, scheduled time, frequency type, and snooze status.
* Given an authenticated user, When they have no upcoming reminders, Then the system MUST return 200 OK with an empty array.
* Given pagination parameters are provided, When the list is long, Then the system MUST support offset/limit pagination.

## Edge Cases

* What happens when a reminder's SnoozedUntil is in the future? It MUST appear in the list sorted by its SnoozedUntil time (not the original ScheduledTime).
* How does the system handle reminders for soft-deleted tasks? They MUST be excluded from the upcoming reminders list.
